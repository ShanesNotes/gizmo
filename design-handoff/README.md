# Gizmo — Design Handoff Package

Complete art-direction overhaul for **Gizmo**. Base direction: **"Lumen & Static"** — *a spark relighting the dark.* Plus a **fusion layer** that weds it to the *Symbolic World* grammar.

## ✦ Fusion layer — "The Lumen Codex" (ancient symbolism × futuristic arcade)
- **`NARRATIVE.md`** — premise/story canon (Gizmo the clanker preserving the spark of humanity vs dehumanized tech, in a gouache cosmos of lost tech). The standalone "neon spark re-illuminates an ancient codex" one-liner is demoted here to a UI/ceremony sub-motif (§5), not the premise.
- **`Fusion-Codex.html`** — open this. The showpiece for the fusion: a neon spark re-illuminating a sleeping codex. Concept, the reframe (Nova→Illumination, Cache→sealed reliquary, the four economies→**covenants**, enemies→counterfeit authority), fused palette (neon light on measured ancient pigments), and the fidelity rules.
- **`FUSION-CODEX.md`** — portable text of the fusion.
- **`HANDOFF-fusion.md`** — paste-ready fused prompts + the Symbolic World's fidelity rules (emulate the hand, translate motifs, no sacred copying).
- **`PROMPT-watercolor-keyart.md`** — locked image-model prompt for the painterly watercolour lane: KA-1 title key art, BG-1 gameplay plate, and CARD-1 illuminated card backers.
- **`PROMPT-warm-system.md`** — locked Claude Design prompt for warming the vector/CSS/SVG system so it composites over the painted lane.
- **`mockup-title-fused.html`**, **`mockup-hud-fused.html`** — the fused screens.
- **`assets-fusion/*.svg`** — gizmo-illuminated, covenant-emblems, cache-reliquary, **emblem** (the chosen Illuminated-G) + **`emblem-mark-mini.svg`** (neon-matched, for tiny sizes), **`rarity-illuminated.svg`** (5-tier rank), enemy-counterfeit + **`enemy-family.svg`** (Mote/Warden/Herald) + **`boss-storm-seal.svg`**.
- **`brand/`** — polished raster emblem + logo lockup (from Grok/Codex image-gen) realizing the chosen Illuminated-G direction.
- **`IMAGE-MODEL-BACKLOG.md`** — prioritized, prompt-ready assets for image-gen models (the phase *after* claude.ai/design).
