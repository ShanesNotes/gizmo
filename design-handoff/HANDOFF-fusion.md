## 0 · Fidelity rules — READ FIRST (from the Symbolic World)

These govern everything. They are non-negotiable.

```
- EMULATE THE HAND, NEVER COPY THE PAGE. The source's artistic execution
  (line, gold application, ornament craft, page apparatus) is the visual ground.
  Literal copying of any source image or story prop = forbidden.
- A SYMBOL IS GAME-READY ONLY WHEN IT CHANGES STATE. Every mark carries a verdict,
  covenant, seal, or scar. No decoration-only ornament.
- TRANSLATE MOTIFS INTO OPERATIONS. No literal apples/towers/hair/crowns as props.
  Use abstract sigils, seals, reliquaries, covenants, thresholds.
- NO SACRED COPYING. No saints, clergy, liturgical text, crosses-as-religious,
  or devotional halos. The "aureole" is a secular geometric energy-roundel only.
- EMBLEM VERDICTS, NOT BADGES. Small marks that telegraph state — never random
  runes, fake sigils, or minimap pins.
- STATE BEFORE LABEL. The object shows what's true before any text.
- MATERIAL MEMORY. Restoration keeps the scar; no clean reset, no spotless victory.
- CHARGED EMPTY SPACE is active — quiet makes the payoff land.
- HANDMADE TENSION — slight asymmetry, imperfect registration, ink grain. Not digital-perfect.

DRIFT TO REJECT (quoting the corpus): rarity glow, victory sparkle, green-smoke evil,
fog/bokeh mysticism, waypoint pins, tutorial popups, corporate cards, tarot polish,
generic parchment stationery, ornamental density without behavior, source-story cosplay.

THE NEGOTIATION (this is a kids' dopamine arcade): joy stays. But —
rarity = gold-leaf + seal density first, glow second; sparkle must RESOLVE into an
illumination (a verdict); glow co-exists with stipple radiance; beautiful gifts may COST.
```

**Lead with a workshop description.** Before designing, study the attached exemplars and write your own account of *the hand* (line, gold application, ornament, page apparatus, light, materiality). Design from that, not from the story.
