# Gizmo

A kid-friendly **bullet-heaven / survivors-like** built in Phaser — *"The Lumen Codex"*: a neon spark re-illuminates an ancient, sleeping codex. Ancient illuminated-manuscript symbolism × futuristic arcade.

## Layout
- `index.html`, `assets/`, `art/` — the playable Phaser build (Vite output).
- `design-handoff/` — the complete art direction + design system: art bible, the Fusion Codex, screen mockups, SVG assets, brand emblems, the image-model backlog, and the Claude Design setup guide. **Start at `design-handoff/README.md`.**

## Notes
- This repo holds the **built game outputs** + the **design system** (pre-build game source is not included here).
- Design system entry points: `design-handoff/Fusion-Codex.html` and `design-handoff/CLAUDE-DESIGN-SETUP.md`.

## Run
Static build — serve the repo root with any static server and open `index.html`:
```
npx serve .
```
