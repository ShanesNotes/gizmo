## Non-Negotiable Scale Law

Map art must prove a true 3D Godot rogue-lite with fixed Diablo-style camera:

- Large walkable landmasses, not tiny chained platforms.
- Broad combat arenas that can support swarms and kiting.
- Long traversal distance with loops, side pockets, and readable landmark return views.
- A first playable slice should imply a 30-minute run, not a 30-second diorama.
- Vertical scenery belongs mainly on edges and landmarks; walkable ground must remain
  readable from the camera.

## Visual Canon Guardrails

- No player-facing wave rounds, countdowns, boss bars, danger meters, or exposure UI.
- Beacon Rekindled is the win condition; HP 0 is the loss condition.
- Saturated teal/cyan is scarce: Gizmo eye/core, guard layer, active Beacon core-spark.
- Sparks are violet-bodied with warm inner glow, not teal or blue mana crystals.
- Face equals alive; faceless equals hollowed.
- Avoid generic gears/circuits as corruption, tiny filigree, generic fantasy boss posing,
  and copyright-adjacent likeness drift.
