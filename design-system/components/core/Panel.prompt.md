The dark-glass "page apparatus" container — gold rule border, inner luminous edge, dashed inner rule. The HUD workhorse for score readouts, build lists, progress.

```jsx
<Panel eyebrow="Covenant">
  <div>Pulse Driver · Spark Magnet · Echo Coil</div>
</Panel>
<Panel tone="danger" eyebrow="The Claiming">…bounty verdict…</Panel>
```

`tone`: `gold` (default) · `danger` (coral border, for bounty/threat) · `plain`. `dashed` toggles the inner rule. Pair with `Seal` corners for full page-apparatus framing.
