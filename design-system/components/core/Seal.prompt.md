Oxblood wax-seal diamond (gold-bordered, rotated 45°) — use as the four page-frame corner marks, or inline as a "sealed / maxed" token. Carries a verdict, never decoration.

```jsx
{/* page-frame corners */}
<Seal style={{ position:'absolute', left:7, top:7 }} />
{/* inline maxed token */}
<Seal size={26} glow>✦</Seal>
```

`children` renders a centered upright glyph. `glow` adds a coral halo for active seals.
