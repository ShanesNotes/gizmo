Build/covenant chip — a hue dot + label + oxblood count badge. The `sealed` variant marks an evolved/locked upgrade with a gold rule and a gold ✦.

```jsx
<Pill dot="var(--clutch)" count={3}>Pulse Driver</Pill>
<Pill dot="var(--flow)" count={2}>Spark Magnet</Pill>
<Pill sealed count="✦">Echo Coil</Pill>
```
